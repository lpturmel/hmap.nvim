# hmap.nvim
Neovim plugin to visualize keymaps

## Installation

Packer
```vim
use {
    'lpturmel/hmap.nvim',
    requires = {'nvim-lua/plenary.nvim'}
}
```

vim-plug
```vim
Plug 'lpturmel/hmap.nvim'
```

*Note: This plugin is not very useful, it was made for learning purposes only (plenary popups, buffers, etc).*
